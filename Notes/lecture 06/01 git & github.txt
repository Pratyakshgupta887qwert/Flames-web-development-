git is basically use for version control system : it is use to store the history of file. 


for checking the username:
git config --global user.name

for checking the email:
git config --global user.email



if you are new user then set up by this command

TO set user name of git globally the command is 
git config --global user.name "name"

TO set user email of git globally the command is 
git config --global user.email "email id"




edit them email and name :
git config --global --edit
